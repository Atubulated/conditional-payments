# Custodex

Trustless payments, settled instantly. Custodex is an enterprise-grade non-custodial escrow protocol built on the Arc Testnet. It secures transactions using conditional logic, cryptographic bonds, and decentralized arbiters.

## Features
* **Timelocked Escrow:** Automated release conditions with cooling-off periods and claim deadlines.
* **Mediated Escrow:** Zero-trust transactions with designated third-party arbiters.
* **Bonded Escrow:** High-stakes agreements requiring up-front collateral from receivers to prevent malicious behavior.

## Tech Stack
* **Frontend:** Next.js (App Router), React, Tailwind CSS
* **Web3 Integration:** Wagmi v2, Viem, RainbowKit
* **Network:** Arc Testnet
